# Google Apps Script Email Webhook

1. Open [script.new](https://script.new)
2. Replace default code with `label-order.gs`
3. Deploy -> New deployment -> Web app
4. Execute as: `Me`
5. Who has access: `Anyone`
6. Copy Web app URL
7. Go to `/admin/dashboard.html` -> `Order Webhook URL` and paste the URL
8. Save settings

After this, label orders from "შექმენი შენი დიზაინი" are emailed to `matsiwine@gmail.com` with PNG attachment.
